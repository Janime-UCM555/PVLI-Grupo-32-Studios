Premisa del juego

Back to the Box es un divertido juego de aventura y decisiones donde volverás a tu pasado gracias a la oportunidad que te brinda un curioso gato con corbata escondido en una caja de cartón. Junto con el y sus cartas rememora el pasado, cambialo y redescubre ¿Volverás a ser el mismo o te devorarán las posibilidades?
